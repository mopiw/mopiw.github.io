美化你的 Windows Terminal

Windows 平台下的终端一直都挺不好使的，比起 Linux 和 MacOS 有不少差距，但是自从微软发布一款船新的终端模拟器：Windows Terminal，它不仅免费开源，还拥有更现代的 GUI 、完整的字体字符 Emoji 渲染机制、GPU 加速和 Fluent 设计风格，最关键它还有丰富的自定义配置。

安装 oh-my-posh 和 posh-git

  1. install oh-my-posh & posh-git 安装 oh-my-posh 和 posh-git

    $ Install-Module posh-git -Scope CurrentUser
$ Install-Module oh-my-posh -Scope CurrentUser

  2. configuration 配置主题

    $ if (!(Test-Path -Path $PROFILE )) { New-Item -Type File -Path $PROFILE -Force }
$ notepad $PROFILE

    在弹出的记事本里输入 3 条记录

    Import-Module posh-git
Import-Module oh-my-posh
Set-Theme Emodipt

Windows Terminal 配置

启动 Windows Termianl，使用快捷键 Ctrl + , 打开配置文件。

{
  "theme": "dark",
  "profiles": {
    "defaults": {
      "fontFace": "JetBrains Mono",
      "background": "#797979",
      "acrylicOpacity": 0.5,
      "useAcrylic": true,
      "fontSize": 13,
      "padding": "8,8,8,8",
      "snapOnInput": true,
      "cursorColor": "#FFFFFF",
      "cursorShape": "bar",
      "startingDirectory": "%USERPROFILE%",
      "backgroundImage": "ms-appdata:///roaming/wallhaven-73yr8e.png",
      "backgroundImageStretchMode": "uniformToFill",
      "backgroundImageOpacity": 0.5,
      "showTerminalTitleInTitlebar": true
    },
    "list": [
      {
        "name": "Windows Terminal @mopi 😎",     //标题
        "commandline": "powershell.exe -nologo", //去掉默认欢迎语
        "suppressApplicationTitle": true,        //支持自定义标题
        "colorScheme": "One Half Dark",          //配色
        "icon": "ms-appdata:///roaming/icon.png" //图标地址
      }
    ]
  },
  "schemes": [
    {
      "name": "One Half Dark",
      "background": "#282C34",
      "black": "#282C34",
      "blue": "#61AFEF",
      "brightBlack": "#282C34",
      "brightBlue": "#61AFEF",
      "brightCyan": "#56B6C2",
      "brightGreen": "#98C379",
      "brightPurple": "#C678DD",
      "brightRed": "#E06C75",
      "brightWhite": "#DCDFE4",
      "brightYellow": "#E5C07B",
      "cyan": "#56B6C2",
      "foreground": "#DCDFE4",
      "green": "#98C379",
      "purple": "#C678DD",
      "red": "#E06C75",
      "white": "#DCDFE4",
      "yellow": "#E5C07B"
    }
  ]
}

 

配置说明

名称 说明
acrylicOpacity 亚克力透明效果大小
useAcrylic 打开亚克力透明效果
snapOnInput 输入的时候自动滚动到输入位置
backgroundImage 背景图片地址 默认本地图片文件夹：C:\Users\{用户名}\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\RoamingState
backgroundImageStretchMode uniformToFill 图片自适应
showTerminalTitleInTitlebar 显示自定义标题
schemes 配色
